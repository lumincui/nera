# Touchpoint Prototype

这是 Nera touchpoint 的可操作通知静态交互原型。

## 目标

- 验证可操作通知如何承载授权类请求、question 请求和完成提醒。
- 将文字 spec 转换为可操作、可感知的交互形态。

## 运行

- 直接用浏览器打开 `index.html`。

## 覆盖场景

- `tool permission request`
- 权限请求说明任务、权限和命令
- 权限请求支持动态选项：`Deny`、`Approve`、`Always`
- `question` 多选
- `question` 选择和文本输入一起提交
- `question` 通过系统听写输入文本
- `idle`
